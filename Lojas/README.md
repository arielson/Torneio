# Pacote para Lojas

Esta pasta reúne os materiais de publicação do app `Torvia` para lojas, separados por plataforma.

Estrutura

- `Android/`
  - textos da Google Play em `.txt`
  - checklist de Data Safety
  - guia de classificação de conteúdo
  - template de notas de versão
  - arquivos auxiliares para publicação
- `iOS/`
  - textos da App Store em `.txt`
  - checklist de App Privacy
  - guia de age rating
  - template de notas de versão
  - arquivos auxiliares para publicação

Arquivos gerados

- `Android/Arquivos/app-icon-1024.png`
- `Android/Arquivos/recurso-grafico-google-play.png`
- `iOS/Arquivos/app-icon-1024.png`
- `iOS/Arquivos/AppIcon.appiconset/` com todos os ícones exigidos no catálogo atual do app

URLs adotadas neste pacote

- Site principal: `https://torvia.ari.net.br`
- Suporte: `https://torvia.ari.net.br/suporte`
- Política de privacidade: `https://torvia.ari.net.br/politica-de-privacidade`
- E-mail de suporte: `arielson.c.ferreira@gmail.com`

Observações

- Os builds (`.aab`, `.apk`, `.ipa`) não foram incluídos.
- Os textos foram padronizados em `.txt` nas pastas de plataforma.
- O recurso gráfico da Google Play foi gerado em `1024x500`.
