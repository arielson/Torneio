# Roteiros Manuais

Esta pasta concentra a base textual dos roteiros manuais da plataforma.

Arquivos atuais:

- `01-admin-geral.md`
- `02-admin-torneio.md`
- `03-fiscal-app.md`
- `04-pescador.md`
- `05-espectador.md`

Uso recomendado:

1. Ajuste o roteiro `.md` desejado quando houver mudancas de produto.
2. Reflita essas mudancas no script `scripts/roteiros/gerar_roteiros_pdf_manual.ps1`.
3. Gere novamente os PDFs para `C:\Users\ariel\Downloads`.

Observacao:

- Os arquivos Markdown servem como fonte de revisao rapida.
- O gerador de PDF continua sendo o script PowerShell, que ja aplica capa, layout e exportacao via Chrome ou Edge headless.
